Design Process

Realizing this idea has proven to be extremely challenging. I researched and applied noise in the transformation process of Kirby. I also attempted to customize the function to make drawing Kirby easier. However, I encountered difficulties when trying to utilize random methods. Although I wrote functions for the transformed status of Kirby, I am unable to incorporate them as assets and use random(). As a result, I sought out tutorials and used random() to generate images that I drew.

Reflection

I learned a lot in this project and I would like to learn more about class and assets to solve the problems that I met. Also, it would be nicer if there were suggestions on how to simplify the code. Besides,it would be better if I can refresh the canvas with keyboards.
